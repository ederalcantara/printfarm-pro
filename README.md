# PrintFarm Pro - Sistema de Gestão para Fazendas de Impressão 3D

## Versão 1.0.0

Sistema web completo para gerenciamento de fazendas de impressão 3D, com:

- **Máquinas**: Cadastro e controle de impressoras 3D
- **Estoque**: Controle de filamentos com alerta de estoque baixo
- **Calculadora**: Cálculo automático de custo + preço de venda
- **Portfólio**: Catálogo de produtos disponíveis
- **Pedidos**: Gestão de ordens de serviço
- **Clientes**: Cadastro de clientes
- **Marketing**: Geração de posts para redes sociais
- **Relatórios**: Estatísticas e análises

## Instalação

1. Instale Python 3.8+
2. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```
3. Execute:
   ```
   python app.py
   ```
4. Acesse: http://localhost:5000
